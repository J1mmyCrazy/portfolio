Digital Image Manipulation Study

A Photoshop study focused on compositing, color grading, adjustment layers, visual experimentation, and image presentation.

Overview

This project explored digital image editing through Photoshop. I experimented with image manipulation, color correction, visual style, and final presentation variations.

Skills Practiced

Photoshop Layer masks Adjustment layers Color grading Compositing Visual design

Process

Original Image

I started with a base image and explored ways to adjust the mood, color, and overall visual impact.

Editing Workflow

I used layers, masks, and adjustment tools to create different visual treatments while keeping the edits organized and editable.

Final Variations

I created multiple final versions to compare different tones, colors, and presentation styles.
