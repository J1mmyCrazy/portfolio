Overview

Typing Survival is a browser-based real-time typing game where players destroy incoming words by typing them correctly. The game becomes harder over time by increasing speed, adding more lanes, and introducing longer words.

Core Features

Real-time typing Enemy word system Dynamic difficulty 1–3 lane progression Combo system Score system Sound effects Background music MongoDB leaderboard Full deployment

Development Breakdown

Gameplay System

I built the core gameplay loop around direct key detection instead of a traditional input box. This kept the focus on the action while allowing the player to type words as they appeared on screen.

Difficulty Progression

The game progresses from Easy to Medium to Hard by increasing word speed, word length, and the number of active lanes. This created a stronger survival feel as the run continued.

Scoring and Combos

I created a scoring system based on difficulty, combo count, words destroyed, and time survived. Perfect words build the combo, while mistakes immediately reset it.

Audio System

I implemented sound effects for correct keys, incorrect keys, completed words, missed words, and background music using the HTML Audio API.

Leaderboard and Deployment

I created a backend API with Node.js and Express, connected the leaderboard to MongoDB Atlas, and deployed the frontend and backend on Render.

Recent Updates

Future Improvements

I would like to add new modes to give players more ways to play beyond the current survival format.

I want to improve the game so it works consistently across more screen sizes and devices.

Future updates could include increasing the challenge of Easy and Medium mode and adding particle effects to make word destruction and player feedback feel more satisfying.

Lessons Learned

This project helped me understand how many systems are involved in making a game feel complete. Beyond the core typing mechanic, I had to think about difficulty, feedback, sound, scoring, deployment, and how to keep the player engaged over multiple runs.
