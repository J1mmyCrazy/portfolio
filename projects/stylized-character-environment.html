Blender • Character Modeling • Environment Design

Stylized Character & Environment Study

A Blender project exploring environment creation, low-poly character modeling, topology, rigging, lighting, and scene composition.

Overview

This project began as a guided Blender environment study and evolved into a larger creative exercise. I expanded the scene by developing an original stylized character concept inspired by an early rhythm-action fighting game idea.

Project Goals

Learn Blender workflow Practice low-poly modeling Explore character design Practice rigging Improve scene composition

Process

Environment Study

I used the environment as a foundation to practice modeling, lighting, camera placement, and final render presentation.

Character Development

I designed a stylized character connected to a musical fighting game concept. The project included early sketches, modeling attempts, redesigns, and final presentation renders.

Rigging & Problem Solving

I created a basic character rig and adjusted the model for posing. When hands became too difficult to model cleanly, I redesigned the character with gloves to simplify the topology while keeping the style consistent.
